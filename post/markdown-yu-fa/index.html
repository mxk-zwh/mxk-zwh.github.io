markdown语法

2023-05-02 1 min read # markdown

&lt;代替大于号，&gt;代替小于号
所有首行缩进

p{
   text-indent: 2em; /*首行缩进*/
}
  • 使用LaTeX的空格符\qquad、\quad、\;、\,控制缩进
    bad: 公式结束符$后紧跟数字将导致公式无法渲染(csdn)
    good:选中时，不会明显看到空格符号被选中
  • 使用Html的空格符&nbsp;、&emsp;控制缩进。
    bad: 选中时，可以明显看到空格符号被选中
    good:在CSDN博客显示时，选中复制不会像LaTeX一样把空格符代码给复制出来，而只会复制出简单的空格符号，粘贴后清除方便
  • 使用Markdown的列表控制缩进。- 1.
  • 使用<figure>标签控制缩进。
<figure>
       1° 子条件1
	<br/>
	2° 子条件2
</figure>
