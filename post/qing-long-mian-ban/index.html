docker

docker run

-d 后台运行容器返回id
-i交互操作
-t终端
-p 主机端口：容器端口
-e 给容器的环境变量
-v （绑定卷） 主机文件夹：容器文件夹
--name 容器叫什么名字
--hostname 容器的hostname
whyour/qinglong:2.11.3 github用户名/镜像名字：标签(版本)
