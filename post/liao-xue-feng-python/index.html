返回函数

比如不需要立即求和，返回求和的函数

def lazy_sum(*args):
    def sum():
        ax = 0
        for n in args:
            ax = ax + n
        return ax
    return sum

模块

大大提高了代码的可维护性，编写代码不必从零开始
经常引用 其他模块，包括Python内置的模块和来自第三方的模块。
把__init__.py看做目录。不是包。

模块名 不要使用中文、特殊字符
2 尽量不要与内置函数名字 冲突。1 命名前看看 Python的所有内置函数（import 就能看）。

函数和变量我们希望给别人使用（public） 正常命名 xxxx
函数和变量我们希望给内部使用 （private） __xxx（两个下划线）
特殊变量，可以被直接引用，有特殊用途 __xxx__
