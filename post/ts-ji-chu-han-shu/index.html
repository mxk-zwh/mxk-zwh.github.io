基本类型

数字类型： 二进制0b1 八进制0o7 十六进制0xf 十进制0

let arr:Array<number>=[1,2]

不明确类型any
js类型：number、string、boolean、null、undefined、object
ts指定类型：any、unknown、never、enum、tuple
指定一种类型
var [变量名] : [类型] = 值;
指定多个类型（联合类型 属于高阶）
var [变量名] : ( [类型] | [类型] )[]
初始没有赋值，默认any任意类型
var [变量名];
[变量名]=1;
初始有赋值 ，指定的类型为 值的类型
var [变量名] = 值;

变量不要使用 name 否则会与 DOM 中的全局 window 对象下的 name 属性出现了重名。

enum

可迭代的 数字+1，默认从0开始，可以自定义（字符串。。）
实战项目大量用到

//编译效果复杂，编译后的东西不需要懂，代码量会比较大，动态类型
// 0,1,2
enum Color {Red,Green,Blue}
//1,2,3 重置起始值
enum Color {Red=1,Green,Blue}
//手动赋值
enum Color {Red=1,Green=2,Blue=4}
//编译效果简单，代码量小，静态
const enum Color {。。。}
