2023-05-20 2 min read

查询 列出 进入 重启 返回

查询已有镜像（显示 镜像id 占用）

docker images -a

列出所有容器（显示 容器id 端口）

docker ps

通过容器id停止镜像

docker stop 容器id

一键停止所有容器

docker stop $(docker ps -aq)
-q只显示所有镜像 容器id

进入镜像内部

docker exec -it 容器名字 bash
docker exec -it 容器名字 /bin/bash

重启镜像

docker restart 镜像名字

删除

通过镜像id删除镜像

docker rmi 镜像Id

通过容器id删除容器

docker rm 容器id

一键删除所有容器

docker rm $(docker ps -aq)

一键删除所有镜像

docker rmi $(docker images -q)

一键删除所有不使用的镜像

docker image prune -f -a

删除所有停止的镜像

docker container prune -f

docker

docker run

-d 后台运行容器返回id
-i交互操作
-t终端
-p 主机端口：容器端口
-e 给容器的环境变量
-v （绑定卷） 主机文件夹：容器文件夹
--name 容器叫什么名字
--hostname 容器的hostname
